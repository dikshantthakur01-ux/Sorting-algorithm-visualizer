# Sorting Algorithm Visualizer

A fully functional web-based visualizer for common sorting algorithms. Built with vanilla HTML5 Canvas, CSS, and JavaScript.

## Features
- **5 Algorithms**: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort
- **Interactive Controls**:
  - Array size: 10-100 elements
  - Animation speed: adjustable delay
  - Play/Pause/Resume sorting
  - Shuffle, Reset, Sort buttons
- **Live Stats**: Comparisons and swaps counter
- **Visual Feedback**: Gradient bars, value labels during sort, responsive design
- **Zero Dependencies**: Pure client-side, works offline

## Quick Start
1. Open `index.html` in any modern web browser
2. Adjust array size and speed using sliders
3. Select an algorithm from dropdown
4. Click **Shuffle** to generate random array
5. Click **Sort** to visualize step-by-step
6. Use **Pause/Resume** during animation

## Demo Command (Windows)
```
start sorting-visualizer/index.html
```

## Technologies
- **HTML5 Canvas** for smooth bar animations
- **CSS3** gradients, backdrop blur, responsive design
- **Vanilla JavaScript** async generators for step-by-step visualization

Works perfectly on desktop, tablet, and mobile devices.
